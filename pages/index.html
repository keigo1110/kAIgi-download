Install Install

Macに入れて、すぐ会議で使う。 Install on your Mac and use it in the next meeting.

  1. 1

    最新版DMGをダウンロード Download the latest DMG

    インストールボタンから kAIgi.dmg をダウンロードします。 Use the install button to download kAIgi.dmg.

  2. 2

    kAIgiを開く Open kAIgi

    kAIgi.dmg を開き、Finderに表示された kAIgi.app を起動します。 Open kAIgi.dmg, then launch kAIgi.app from the Finder window.

  3. 3

    Applicationsへ移動して開く Move to Applications

    確認画面で「移動して開く」を選ぶと、コピー先のアプリが自動で開き直されます。 Choose “Move and Open” when prompted. kAIgi copies itself to Applications and reopens from there.

  4. 4

    初回設定を済ませる Finish first-run setup

    会議音声を扱うためのmacOS確認と、AI Research用のOpenAI API key設定をアプリ内で済ませます。 Complete the macOS prompts for meeting audio and add your OpenAI API key for AI Research inside the app.

Help when needed Help when needed

困った時だけ開いて確認。 Open these only when something needs attention.

Micの許可がうまくいかない Microphone access is not working

kAIgiを終了し、Applications内の kAIgi.app から起動し直してください。一度拒否している場合は、システム設定のマイクでkAIgiをオンにします。 Quit kAIgi and reopen kAIgi.app from Applications. If access was denied once, enable kAIgi in macOS Microphone settings.

内部音声が入らない System audio is missing

システム設定の画面収録または画面とシステムオーディオ収録でkAIgiがオンになっているか確認し、変更後にアプリを再起動します。 Enable kAIgi in Screen Recording or Screen & System Audio Recording settings, then restart the app after changing the permission.

AI Researchが動かない AI Research is not producing cards

設定でAI Researchを有効にし、OpenAI API keyが保存済みか確認してください。議論内容が少ない時は、カードが出るまで少し発話が必要です。 Enable AI Research in settings and confirm that your OpenAI API key is saved. If the discussion is still short, kAIgi may wait for more context before creating cards.

相手の声が二重に入る The other speaker is captured twice

スピーカー再生音がマイクにも入っている可能性があります。会議ではヘッドホンまたはイヤホンの使用を推奨します。 Your microphone may be picking up speaker output. Use headphones or earbuds during meetings.
