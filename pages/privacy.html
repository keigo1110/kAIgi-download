kAIgi Privacy Policy

kAIgiは会議音声を扱うアプリです。録音対象、保存場所、任意の外部送信について、ユーザーが判断できるように明確に説明します。

Last updated: 2026-06-28

収集または保存するデータ

kAIgiは、ユーザーが録音を開始したセッションについて、Mac上で再生されている内部音声、マイク音声、文字起こし、セッションメタデータを保存します。 これらは標準では ~/Documents/KAIgiRecordings に保存されます。

ローカル文字起こし

文字起こしにはApple Speechのオンデバイス認識を使用します。クラウドSTT APIへの自動フォールバックは行いません。

OpenAI連携

Research機能を有効にした場合、匿名化した直近の文字起こしと検索論点をOpenAI APIへ送信することがあります。 この機能は任意であり、ユーザーがOpenAI API keyを設定した場合のみ使用されます。API keyはmacOS Keychainに保存されます。

会議参加者への通知

会議、通話、対面の会話を録音する場合は、録音を開始する前に参加者へ通知し、必要な同意を得てください。 録音に関する法令や社内ルールは地域や組織により異なります。

データの削除

保存済みの録音と文字起こしは、Finderから該当セッションフォルダを削除することで削除できます。 OpenAI API keyはアプリ内の設定から削除できます。

問い合わせ

問題や質問はGitHub Issuesで受け付けます。 https://github.com/keigo1110/kAIgi-download/issues
