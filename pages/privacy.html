kAIgi Privacy Policy kAIgi Privacy Policy

kAIgiは会議音声とAI Researchを扱うアプリです。録音対象、保存場所、任意の外部送信について、ユーザーが判断できるように説明します。 kAIgi handles meeting audio and AI Research. This policy explains what is recorded, where data is stored, and when external services may be used.

Last updated: 2026-06-29

保存するデータ Data stored by kAIgi

kAIgiは、ユーザーが録音を開始したセッションについて、Mac上で再生されている内部音声、マイク音声、文字起こし、AI Researchのカード、セッションメタデータを保存します。これらは標準では ~/Documents/KAIgiRecordings に保存されます。 For sessions started by the user, kAIgi stores system audio, microphone audio, transcripts, AI Research cards, and session metadata. By default, these files are stored in ~/Documents/KAIgiRecordings.

文字起こし Transcription

文字起こしにはApple Speechのオンデバイス認識を使用します。クラウドSTT APIへの自動フォールバックは行いません。 kAIgi uses Apple Speech on-device recognition for transcription and does not automatically fall back to a cloud speech-to-text API.

AI ResearchとOpenAI連携 AI Research and OpenAI

AI Research機能を有効にした場合、匿名化した直近の文字起こしと検索論点をOpenAI APIへ送信することがあります。この機能は任意であり、ユーザーが設定したOpenAI API keyを使用します。API keyはmacOS Keychainに保存されます。利用モデルはkAIgiの既定値を使用し、ユーザーがアプリ内でモデルを選択する必要はありません。 When AI Research is enabled, kAIgi may send sanitized transcript context and research prompts to the OpenAI API. This feature is optional and uses the OpenAI API key entered by the user. The API key is stored in macOS Keychain. kAIgi uses its default model configuration; users do not need to choose models in the app.

会議参加者への通知 Notifying meeting participants

会議、通話、対面の会話を録音する場合は、録音を開始する前に参加者へ通知し、必要な同意を得てください。録音に関する法令や社内ルールは地域や組織により異なります。 Before recording a meeting, call, or in-person conversation, notify participants and obtain any required consent. Recording rules vary by location and organization.

データの削除 Deleting data

保存済みの録音、文字起こし、AI Researchの結果は、Finderから該当セッションフォルダを削除することで削除できます。OpenAI API keyはアプリ内の設定から削除できます。 Saved recordings, transcripts, and AI Research results can be deleted by removing the relevant session folder in Finder. The OpenAI API key can be removed from the app settings.
